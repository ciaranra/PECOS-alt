# Concepts

This section covers the physics, mathematics, and algorithms behind PECOS. It is
intended as pedagogical material for understanding *why* things work the way they do,
rather than *how* to use the API.

## Simulation

- **[Clifford+RZ Simulator](clifford-rz-simulator.md)**: Sum-over-Cliffords simulation
  using CH-form stabilizer states. Covers the CH-form representation, how RZ gates
  create terms, measurement via inner products, and the optimizations (Clifford frames,
  RZ fusion, sparse matrices, structural sharing).
